# 在线问卷调查系统

## 功能特性
- 🎯 可视化问卷设计
- 👥 实时协作编辑
- 🤖 AI智能题目建议
- 📱 响应式设计
- 💾 本地数据存储

## 支持的题型
1. 单选题 (radio)
2. 多选题 (checkbox) 
3. 填空题 (text)
4. 文件上传题 (file)

## 技术栈
- React 18
- PeerJS (P2P协作)
- 阿里云通义千问 (AI建议)

## 安装运行
```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建生产版本
npm run build
